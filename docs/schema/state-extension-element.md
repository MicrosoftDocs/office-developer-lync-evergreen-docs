# state/extension element


**Applies to:** Lync 2013 | Lync Server 2013

Specifies an application-specific extension to a presence state element.

```xml
<st:extension xmlns:st="http://schemas.microsoft.com/2006/09/sip/state" >
   <[any]>any element</[any]>
</st:extension>
```

st:extensionType
