# services/service/capabilities/isfInk element


**Applies to:** Lync 2013 | Lync Server 2013

Specifies the capability for a presentity to handle images of the ISF format.

```xml
<capabilities uri="uri" preferred="boolean" [anyAttr]="anyattr" 
              xmlns="http://schemas.microsoft.com/2006/09/sip/service">
     <isfInk 
          preferred="boolean" 
          preferredEndpointId="endpointId"
          uri="uri" 
          render="boolean" 
          capture="boolean" 
          publish="boolean"
          version="uint" 
          deviceAvailability="uint" 
          anyAttr="anyattr" />
</capabilities>
```

## Example

The following XML code snippet describes a service capability to handle ISF-formatted images for the user, sip:alice@contoso.com.

    <se:service xmlns:se="http://schemas.microsoft.com/2006/09/sip/service">
      <se:capabilities>
        <se:isfInk uri="sip:alice@contoso.com" render="true" publish="true" />
      </se:capabilities>
    </se:service>
