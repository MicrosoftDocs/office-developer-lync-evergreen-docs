# rolePrecedence element


**Applies to:** Lync 2013 | Lync Server 2013

Specifies the precedence hierarchy of roles when a member is assigned multiple roles.

```xml
<rolePrecedence >
   < rolePrecedenceEntry >...</ rolePrecedenceEntry >
</ rolePrecedence >
```

rolePrecedenceType
