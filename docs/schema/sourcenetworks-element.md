# sourceNetworks element


**Applies to**: Lync Server 2013

Specifies the networks to which a particular member belongs.

```xml
<sourceNetworks>
   < sourceNetwork>...</ sourceNetwork>
</ sourceNetworks >
```

sourceNetworksType
